# [Gifts in Inbox](https://www.mousehuntgame.com/preferences.php?tab=mousehunt-improved-settings#mousehunt-improved-settings-feature-gifts-in-inbox)

Adds recently discovered MouseHunt gift links to a "Gift Links" tab in the inbox for easy claiming.

## Features

- Adds a "Gift Links" tab to the inbox that lists recently discovered gift links, with the gift name, who found it, and when it expires
- Claim a gift directly from the list with a single click
- Tracks which links you've claimed or hidden so the list stays manageable
- Claimed gift buttons stay clickable as a link to the gift page, and failed claims offer a "Mark as claimed" link
