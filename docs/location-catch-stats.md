# [Location Catch Stats](https://www.mousehuntgame.com/preferences.php?tab=mousehunt-improved-settings#mousehunt-improved-settings-feature-location-catch-stats)

Adds a "Location Catch Stats" option to the Mice menu to see your catch stats for the current location.

## Options

- **Color rows based on crown**: Colors each row based on the crown you have for that mouse.
- **Show number of misses**: Also shows how many times each mouse has missed you.
